## Transport

- TCP loopback `127.0.0.1:7437` (override via `MINI_HOST` / `MINI_PORT`)
- Each message is one `\n`-terminated JSON line (NDJSON)
- Commands use JSON-RPC 2.0; Events use a custom `kind=event` envelope
